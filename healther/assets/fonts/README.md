# Fontes Italianno

La police Italianno est utilisée dans l'application HEALTHER pour le logo et le titre stylisé.

**Emplacement :** `assets/fonts/Italianno/Italianno-Regular.ttf`

**Configuration :** La police est configurée dans `pubspec.yaml` et utilisée dans le widget `HealtherLogo`.

**Note :** Si la police Italianno n'est pas disponible, l'application utilisera automatiquement une police serif comme fallback. Les animations et effets visuels fonctionnent indépendamment de la police utilisée.
